Mapping Wikileaks' Cablegate thematics using Python, MongoDB, Neo4j and Gephi

FOSDEM's data devroom, Brussels, Feb 5 2011

Goals

  • We want to see the how the discussions within the cables are composed and relate to each other

  • Extraction from cables' contents to produce the network

  • Use meta-data (tags, origin, date) as they are and not as a structure

produce a mixed occurrence and cooccurrence network of keywords and cables

How ?

  • analyse the ~2600 full-texts with Python, and a set of productive librairies

  • use MongoDB for document storage and Neo4j for network storage

  • explore the network with Gephi

Import cables to MongoDB

  • BeautifulSoup : html made easy

    • automatically decodes to Python's unicode
    • quietly handles html's tree errors
    • traversing methods as Python methods and lists, becoming comparable to Javascript
    • higher performances with the SoupStrainer class : tells the parser to target a specific part of the document

  • NLTK : cleaning html

    • one effort though : handle decoding manually
    • Python provides safe built-in functions to manage that

  • MongoDB : document storage

    • inserting and reading records as Python dictionaries transparenly
    • types support : unicode, nested lists and dictionaries, datetime

  • Hopefully, Wikileak's archive respects simple structure, making data hackers' job easy !

Extract keyphrases

  • decompose text into words : nltk.sent_tokenize and nltk.TreebankWordTokenizer

  • Part of Speech tagging with nltk.tag
    • Compose and save a tagger object with nltk-trainer
    • high quality POS tagging, composed of multiple algorithms and trained against NLTK's integrated pre-tagged corpora

  • group keyphrases using stemming: nltk.PorterStemmer

Choose relevant keyphrases

  • automatic selection of keyphrases of 1, 2 or 3 consecutive words: a set of homemade and built-in filters

    • Homemade POS tag regexp filtering useless series of words

    (VB,|VBD,|VBG,|VBN,|CD.?,|JJ.?,|\?,){0,2}?
    (N.?,|\?,)+?(CD.,)?

    )+?
